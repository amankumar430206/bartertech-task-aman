# Transaction Insights Dashboard

A modern, responsive financial transaction dashboard built with **Next.js 14 (App Router)**, **TypeScript**, **TanStack Query (React Query)**, and **Tailwind CSS**.

Features infinite scrolling, real-time filters, debounced search, client-side aggregations, and clean architecture separation.

https://github.com/amankumar430206/bartertech-task-aman

## ✨ Features

- Infinite scrolling with pagination (MockAPI)
- Debounced search by name / transaction ID
- Multi-select status filter
- Real-time derived insights:
  - Total transactions
  - Total successful amount
  - Success rate (%)
  - Top spending category
- Responsive design (mobile + desktop)
- Skeleton loaders & loading states
- Error handling & clean component structure
- No external UI libraries — pure Tailwind + custom primitives
